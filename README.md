# WatchLister Project

WatchLister: A website for all things movie-related!

WatchLister lets users discover films, save their favorites, and rate what they’ve watched using a 5-star system. 
It includes user authentication, a personalized profile, and an interactive interface where movies can be browsed, rated and organized for easy access.

## Setup

1. Import `database.sql` into your MySQL database
2. Update database connection in `/backend/config.php`
